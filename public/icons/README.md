# App Icons

Place the following files in this directory:

- `icon-192.png` — 192×192 px
- `icon-512.png` — 512×512 px

Both are referenced in `public/manifest.json` and used as the PWA home screen icon.

**Suggested generator:** https://realfavicongenerator.net

Use a trophy emoji or icon in gold `#C9A84C` on a dark `#0B0D18` background.
